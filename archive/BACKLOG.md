# Archetype - Feature Backlog

Obsidian plugin for active learning through speed reading and touch typing.

## Vision

All-in-one tool for:
* Speed reading training
* Touch typing training  
* Knowledge retention through active engagement
* Dual-channel learning (visual input + motor output)

## Planned Features

### Touch Typing Progression Mode
Chunk text and require typed response to advance:
* Display chunk
* Wait for typed response that matches
* Validation modes:
    * Exact match
    * Case-insensitive match
    * Fuzzy match (allow minor typos)
    * Semantic match (LLM-based, future)

See [specs/003-touch-typing-progression/spec.md](specs/003-touch-typing-progression/spec.md) for full design.

### Additional Planned Features
* Pause/resume controls
* Progress indicators
* Reading statistics and metrics
* Keyboard shortcuts
* Skip forward/backward
* Adjustable overlay opacity
* Custom color schemes
